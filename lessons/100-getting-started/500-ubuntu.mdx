<h5>Overview</h5>
One of the most popular Linux distributions for personal machines is Ubuntu. Ubuntu
also releases its own desktop environment manager Unity by default.

<h5>Package Management</h5>
Ubuntu is a Debian-based operating system developed by Canonical. So it uses a core
Debian package management system.
